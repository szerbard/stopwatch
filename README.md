# Stopwatch

Precision stopwatch with lap times, splits, and fastest/slowest tracking.

## How to Use

**Step 1:** Open `index.html` in your browser.

**Step 2:** An onboarding tutorial appears on first visit. Click **Open Stopwatch** (or Skip).

**Step 3:** Click the green **Start** button to begin timing.

**Step 4:** Click **Lap** while running to record a split time.

**Step 5:** Each lap shows the time since the previous lap. Fastest = green, Slowest = red.

**Step 6:** Click **Pause** (red button) to stop the timer temporarily.

**Step 7:** Click **Reset** to clear everything and start over.

## Features
- Millisecond precision timing
- Lap recording with split times
- Fastest/slowest lap highlighting
- Start/Pause/Lap/Reset controls
- Monospace digital clock display

## Tech
HTML + CSS + JavaScript | Date API
